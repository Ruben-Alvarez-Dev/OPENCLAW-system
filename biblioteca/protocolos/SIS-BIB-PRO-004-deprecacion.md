# Protocolo de Deprecación

**Nivel:** Sistema
**Dominio:** Biblioteca
**Tipo:** Protocolo
**Versión:** 2.1.0

---

## Propósito

Gestionar el ciclo de vida de componentes obsoletos del sistema de manera segura y controlada.

---

## Estados del Ciclo de Vida

```
ACTIVO → DEPRECATED → SUNSET → REMOVED
```

---

## Proceso

1. **Anuncio** - Notificar afectados
2. **Período de Gracia** (90 días) - Warnings activos
3. **Sunset** (30 días) - Degradar funcionalidad
4. **Eliminación** - Archivar histórico
